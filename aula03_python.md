 * exemplo JAVA:
  
 ```java  
 if (condicao logica) {
     codigo
     codigo
     codigo
 }
 else if (condicao logica) {
     codigo 
     codigo 
     codigo
 }|
 else (condicao logica) {
     codigo
     codigo
 }
```  
 * exemplo PYTHON:

```python
if (condicao logica)
    codigo

    codigo
else if (codicao logica)
```
