## introdução a python

### aula 4, parte 1, Estruturas de repetição
* while
    * exemplo: 
    ```python
    count = 1
    while count <= 5:
        print (count)
        count += 1
    ```
* for
    * exemplo:
    ```python 
    for n range (0,10, 1):
        print (n)
    ```
    * range (inicio, parada, incremento) 
    
* interrupções 
    * para interromper o laço _break_, para reiniciar _continue_
